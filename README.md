# Naruto-website 🍃

A futuristic, interactive fan website dedicated to the legendary anime **Naruto**. Featuring a sleek cyberpunk design with a comprehensive Shinobi Database, story arcs, character information, and inspirational quotes from the series.

## 🌟 Features

- **Futuristic UI Design** - Neon-themed interface with cyan and orange color scheme
- **Responsive Layout** - Fully optimized for desktop and mobile devices
- **Interactive Navigation** - Smooth single-page application (SPA) routing
- **Hero Section** - Eye-catching landing page with animated kanji
- **Story Arcs** - Browse through major story arcs from the series
- **Shinobi Database** - Searchable character database with filtering options
- **Quote Section** - Animated film reel displaying iconic character quotes
- **About Section** - Learn about the Naruto series with timeline
- **Smooth Animations** - Custom animations including scanlines, glow effects, and transitions

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Advanced styling with:
  - CSS Grid & Flexbox
  - CSS Variables for theming
  - Keyframe animations
  - Backdrop filters & effects
  - Clip-path for futuristic shapes
- **Google Fonts** - 
  - Orbitron (futuristic headings)
  - Noto Serif JP (Japanese text)
  - Rajdhani (body text)

## 🚀 Quick Start

### Option 1: Open Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/muhammedzayinp/Naruto-website.git
   ```

2. Open `Naruto.html` in your browser:
   ```bash
   cd Naruto-website
   open Naruto.html  # macOS
   # or
   start Naruto.html # Windows
   ```

### Option 2: View Online
Simply visit the deployed version (if hosted) and navigate through the site.

## 💡 Tips for Customization

### Change Colors
Edit the CSS variables in the `<style>` section:
```css
:root {
    --orange: #ff6b00;
    --cyan: #00d4ff;
    --dark: #03030a;
    /* ... more variables ... */
}
```

### Modify Fonts
Update Google Fonts imports and font-family declarations in CSS.

### Add New Characters
Update the character data and cards in the database section.
